# 项目描述
我想做一个可以查看和分析用户埋点以及项目报错的项目
目标用户主要是项目的前端开发和产品经理

这个项目主要有埋点，漏斗，错误监控功能。
# 埋点功能
可以收集用户的点击等事件并为产品体验进行分析
# 漏斗功能
可以帮助分析是在哪一步用户流失的比较多。
